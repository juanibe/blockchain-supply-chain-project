# Supply chain project

Descentralized application to track item creation and selling process

### How to run

- Clone the repository
- Run npm install
- Add environment variables: IFURA_SEPOLIA_URL, INFURA_API_KEY, MNEMONIC in a .env file

### Tests

- Run truffle test
